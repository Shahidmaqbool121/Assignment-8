Explain the difference between absloute and relative positioning

Absloute Relative
the element is positioned absolutely to its first positioned parent. the element is positioned relative to its normal position.
Absolute positioning allows you to place your element precisely where you want it. When you set the position relative to an element, without adding any other positioning attributes (top, bottom, right, left) nothing will happen.
Absolute positioning is done relative to the first relatively (or absolutely) positioned parent element When you add an additional position apart from relative, such as left: 20px the element will move 20px to the right from its normal position.
